ElectroBullet

PushBullet is a great service that allows one to send and recive SMS messages on thier computer.
Unfortunately they didn't include a mac app so I wrote one...

In order to use this program you will need to:

Download it here, unzip it and run it as you would any other application.

Install the PushBullet app on your phone, either iPhone or Android.

The first time you run ElectroBullet it will open a browser to the PushBullet OAuth page for authorization. Upon authorization ElectroBullet will restart and be fully functional.
