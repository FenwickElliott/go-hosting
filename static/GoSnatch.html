GoSnatch

A simple Go script that consumes the Spotify API to add the currently playing song to a designated playlist.

Usage:

To install: "go get github.com/fenwickelliott/snatch"

To use: "snatch"

How it works:

If this is the first time GoSnatch is run it must initialize itself. To do this it opens a browser to Spotify's OAuth page. If the user grants access their page redirects to a temporary local server which harvest the access code and closes it'self down.

The access token is then exchagned for an access token and a refresh token at the Spotify API. These tokens allow GoSnatch to access your Spotify acount and refresh when needed.

Once initialized GoSnatch quereys the Spotify API for the now playing track and records it's id. It simultaniously looks for a playlist called 'GoSnatch' and creates one if it can't be found. If the playlist does not already contain the currently playing track it is added with a final REST call.

Upon completion a native MacOs notification is sent giving the result.
